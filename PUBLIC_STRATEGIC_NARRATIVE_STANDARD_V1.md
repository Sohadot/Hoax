# Public Strategic Narrative Standard v1

## Scope

Governs five strategic narrative routes added in Sprint 97 (DEC-115).

## Required Components

Each narrative page must include:

1. Reference summary
2. Narrative purpose
3. Category frame
4. Strategic thesis
5. Reference Answer block
6. Source Confidence block
7. Cite This Reference block
8. Retrieval Capsule
9. Page-end reference navigation
10. Boundary reminder

## Depth

Minimum 850 visible words per route.

## Anchors

Stable IDs: `reference-summary`, `narrative-purpose`, `category-frame`, `strategic-thesis`, `reference-answer`, `source-confidence`, `cite-this-reference`, `retrieval-capsule`, `boundary`.

## Linking

- Homepage link
- `/entry-points/` link
- At least 3 utility, pathway, or entry-point routes
- At least 5 reference routes
- At least 2 sibling narrative routes

## Prohibited

Upload, scoring, verdict, detector claim, public API, automated report, JavaScript, forms, chatbot, generator, real-world case evaluation.
