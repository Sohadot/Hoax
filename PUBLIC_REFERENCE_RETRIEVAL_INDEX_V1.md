# Public Reference Retrieval Index v1

**Decision:** DEC-136  
**Sprint:** Sprint 118  
**Status:** Adopted — visible production route

## Route governed by this decision

This sprint adds one visible production route:

- `/retrieval-index/` (ROUTE-0102, PUB-FILE-0102)

DEC-136 governs this route directly in Sprint 118.

## Governing sentence

The retrieval index organizes public reference routes for citation-safe retrieval; it does not search, rank, score, summarize, generate, verify, detect, assess cases, or answer cases.

## Purpose

Public Reference Retrieval Index v1 organizes existing Hoax.ai routes by retrieval intent so humans and AI agents can identify companion references without route-by-guessing.

## Retrieval intents covered

1. Thesis and category framing
2. Evidence-risk concepts
3. Evidence condition definitions
4. Crosswalk and relation retrieval
5. Reading sequence orientation
6. Audience-specific orientation
7. Boundary and non-detector language
8. Citation and retrieval blocks
9. Strategic review and category-asset understanding
10. System-level navigation

## Boundary

This route is reference infrastructure only. It does not introduce search interface behavior, ranking outputs, generated answers, verification outputs, detector outputs, case-assessment outputs, dashboard behavior, API behavior, upload behavior, workflow behavior, report generation behavior, or transaction behavior.

## Production-first compliance

This governance file is tied to the visible Sprint 118 production route `/retrieval-index/`. It is not abstract preparatory governance.
