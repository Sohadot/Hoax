Readiness purpose

An asset that hides its value behind login walls or governance PDFs alone is hard to diligence. Hoax.ai publishes reference value on indexable static routes.

This page maps how utility pages teach manual application, reference pages stabilize concepts, pathway pages route by evidence condition, entry points route by role, and narrative pages explain category frame.

Citation blocks, Source Confidence sections, Reference Answer units, and Retrieval Capsules appear across the surface so humans and agents can quote safely.
