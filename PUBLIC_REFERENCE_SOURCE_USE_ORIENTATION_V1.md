## Governing sentence

Source use orientation explains how Hoax.ai uses sources to support reference boundaries and public claims; it does not create proof, verification, authority certification, endorsement, case conclusions, detector evidence, score basis, legal source status, or academic source status.

## Purpose

Public Reference Source Use Orientation v1 teaches how sources function inside Hoax.ai as support for public reference claims, source-confidence framing, claim-source traceability, and citation-safe context.

## Source use orientation themes

1. Source use inside Hoax.ai
2. Sources as reference support
3. Source Confidence blocks
4. Evidence ledger traceability
5. Claim-source relationship
6. Source use and citation orientation
7. Source use and AI agents
8. Source use without proof language
9. Source use without detector language
10. Source use and strategic review

## Boundary

This route is source-use orientation only. It does not introduce proof claims, verification claims, authority certification, source endorsement, legal or academic source status, detector evidence, score basis, verdict support, source database interfaces, source authority directories, workflow behavior, API behavior, upload behavior, form behavior, report generation behavior, dashboard behavior, or transaction behavior.
